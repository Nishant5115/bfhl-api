# BFHL API

Backend REST API built for the Chitkara University Qualifier 1.

## Endpoints
- **GET /health** – Server health check  
- **POST /bfhl** – Handles `fibonacci`, `prime`, `lcm`, `hcf`, and `AI`

## Tech Stack
Node.js, Express.js, Axios, Google Gemini API

## Live URL
https://bfhl-api-8gaa.onrender.com
